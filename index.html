Use With F-Droid

  1. Open the F-Droid client.
  2. Add a repository using this URL: https://<your-github-pages-domain>/repo
  3. Refresh repositories to fetch app metadata and signatures.
F-Droid clients require repository metadata files (such as index files) inside repo/. If apps do not appear in the client, run fdroid update before publishing.
